Game data

Each game session is identified by an server-unique room name you choose when you create it. A room name will be visible in the URL, but is not publically listed or shared by us. However, everyone who knows that name or can guess it, can enter the room.

All data in a room - the pieces, the game board and everything else on it - is stored separate from other room and other servers. Only those that have entered a particular room on a particular server can see/access that data.

Rooms do not store information about the players, only the current state of its tables' layout, the pieces and its library / assets. Rooms also do not store the information mentioned above under Access data.

All data in a room is shared with all people that enter it. Everyone in a room can download some or all this data at once (so-called snapshots) at any time.

Rooms and all their data are deleted from this server after a certain period of inactivity of any table. See the information below the join screen for the actual value of this period - usually 48 hours. If everyone leaves a room and nobody enters it again for that period, it and all contained game data & assets are automatically deleted.

After a room is deleted, its name is free again for others to use. However, no data is taken over from the previous room with the same name.

Snapshots

A snapshot is a .zip file containing the state of a room at a certain point in time. This includes the game setup and all assets. Those that have entered a room can download such a snapshot to their local computer at any time.

While we delete rooms after a period of inactivity (see above), we can not exert any control over downloaded snapshots by their previous players. They can recreate rooms from those snapshots, either on this or any other FreeBeeGee server, by uploading them again at any time.
