Unfortunately, this webserver can't run FreeBeeGee (yet). This can have the following reasons:

  • .htaccess support might be disabled. Check your webserver's config file or your provider's admin-interface and enable this feature. It is also called AllowOverrides.
  • FreeBeeGee files might be missing. .htaccess files are considered hidden files on some operating systems. Verify that all files and directories from the .zip/.tar.gz are on the server and there are no other files there. Also verify the file-permissions for the web-server.
  • This might not be an required Apache HTTP Server.

If none of the above helps, take a look in the servers' error.log for clues.
